Detailed Features

Here are the key features of All in One Dynamic Dashboard.
Facilitates Odoo - Stripe Payment Gateway Integration
  • Allows quick one-step integration of Odoo with Strip Payment Gateway.
  • A Stripe merchant account is necessary to get the integration credentials.
  • Click here to know more >> https://dashboard.stripe.com/register
Redirect to Stipe portal for secure Online Payments
  • The module redirects the customers to the Stripe Portal to ensure secure checkout.
  • The payment is completed on Stripe Portal.
  • Once done, the customer is directed back to the Odoo Website to continue shopping. 
Offers a secure and safe payment method on Odoo Website
  • Stripe is completely compliant with PSD2 and is SCA enabled.
  • These directives aim at improving transparency and avoid fraud.
  • They also help in improving the security while making transactions with Stripe. 
3D Secure enabled for enhanced security
  • The module comes with 3D-secure feature enabled for Odoo Website.
  • 3D secure enables a secondary security check to authenticate payer via OTP.
  • The feature makes the payment more secure as it involves double verification.
Saves Odoo - Stripe Transactions in backend
  • The module saves the transaction details at Odoo backend for each order.
  • The admin can see them in the order details in the backend.
Facility for customers to process check out from my account section
  • The customer can pay directly at the time of checkout with Stripe on Odoo website.
  • Alternatively, the payment for a quotation can be made from My account section. 
Option To Save Customer Cards For Quicker Checkout
  • The customer can save their cards directly on Stripe Server.
  • The saved cards can be then used while making the payment on Odoo website.
  • It allows for a quicker checkout for customers on Odoo website.
